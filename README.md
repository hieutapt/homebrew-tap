# homebrew-tap

Homebrew tap for [`seal-cli`](https://github.com/hieutapt/seals-subscription-cli) — a CLI for the [Seal Subscriptions](https://www.sealsubscriptions.com) Merchant API.

## Install

```bash
brew tap hieutapt/tap
brew install seal-cli
```

## Usage

```bash
# Set up your API token
seal-cli profile set --name default --token YOUR_SEAL_TOKEN

# List subscriptions
seal-cli subscription list

# Get, cancel, pause, resume
seal-cli subscription get 12345
seal-cli subscription cancel 12345
seal-cli subscription pause 12345
seal-cli subscription resume 12345

# Manage billing attempts
seal-cli billing-attempt skip --id 123 --subscription-id 456
seal-cli billing-attempt reschedule --id 123 --subscription-id 456 \
  --date 2025-12-01 --time 14:30 --timezone "+00:00"

# Output raw JSON for scripting
seal-cli subscription list --json
```

Full documentation: [github.com/hieutapt/seals-subscription-cli](https://github.com/hieutapt/seals-subscription-cli)
