# homebrew-tap

Homebrew tap for `seal-cli` — a CLI for the [Seal Subscriptions](https://www.sealsubscriptions.com) Merchant API.

## Install

```bash
brew tap hieu/tap
brew install seal-cli
```

> Replace `hieu` with your GitHub username.

The `Formula/seal-cli.rb` file in this repo is **automatically updated** by GoReleaser on every tagged release of [seals-subscription-cli](https://github.com/hieu/seals-subscription-cli).
